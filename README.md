# 半导体研究知识库 · Semiconductor Atlas

一个长期化、可交互的中文半导体专题知识库，分**两条轨道**。纯前端、单文件页面、无外部依赖、无需联网。

## 🔗 在线访问
GitHub Pages：<https://jxnchj.github.io/semiconductor-lab/>

进入是**选轨门户**（`index.html`），选一条轨道：

- **第一轨 · 原理科普** → `principles/` —— 芯片制造原理**交互图解**：晶体管 / 反型层、光刻（含光刻胶感光机制）、先进制程（FinFET · GAA · CFET · 背面供电）、制造流程（含 CMP、电子特气）、硅片·掩膜（含外延片）、存储器、先进封装、功率 / 化合物半导体。理解"芯片是怎么造出来的"。
- **第二轨 · 产业链价值** → `atlas.html` —— **研究终端**，覆盖**六大赛道**（材料 / 设备 / 晶圆制造 / 封装测试 / EDA·设计 / 存储器）。每个赛道 7 个子页：定位 / 细分 / 价值量 / 护城河 / 玩家格局 / 驱动与风险 / 数据·来源。配**全局搜索**与**上市公司速查**。投研视角，数据**标来源与时间**、区分「事实 vs 厂商叙事」。
