## 为什么重构
现有平铺的 ①–⑩ 标签**混了 4 条分类轴**（工序/器件构造/品类/上游+封装），且：先进制程内部是"节点经济＋器件结构"杂烩；光刻是制造流程的子工序被单拎；后道封测＋先进封装本是同类。

## 目标结构（4 组 · 两级导航）
```
A · 导览（看懂全局）
   1. 全链条地图（按工序切的地图）          [现 s-chain]
   2. 基础概念 · 术语图鉴                    [现 s-basics]
B · 器件（芯片靠什么工作）
   3. 晶体管原理（开关/反型层/MOSFET/CMOS）  [现 s-tr]
   4. 先进结构演进（短沟道→FinFET→GAA→CFET→背面供电→3D）  ← 从"先进制程"拆出的【结构段】
C · 制造（怎么造，沿链条：上游→前道→后道）
   5. 硅片·掩膜（上游原料）                  [现 s-wafer]
   6. 制造流程（前道 FEOL/BEOL）            [现 s-fab]
   7. 光刻（关键工序）                       [现 s-litho]

```

## 关键改造 1：拆「先进制程」(s-adv) 成两页

- **结构段 → 新页 `s-advstruct`「先进结构演进」**，放 B 组、接在晶体管后。
- **节点经济段 → 留在 `s-adv`，改名「制程节点与经济」**，放 C 组。
- 注意保留交互：密度网格 `#ngrid`/`.nbtn`、CFET/背面供电 SVG、3D 那段。chiplet 过渡指向「先进封装」。
- 拆时把两段之间的"过渡 pill/note"重写，使各自成篇。
