## 3. 第二轨目标 & 范围
- 目标：一张**产业链价值地图** + 若干**细分赛道深度研究页**。
- 视角：投研 / 产业，尽量贴近真实——份额、金额、公司给真实数据，**标注来源与时间**，并区分「事实 vs 厂商叙事」。

## 3.5 分轨原则 & 轨1回补接口（重要）
**按"视角"分轨，不按"主题"——同一材料可两轨都出现、不重复：**
- 机制 / 原理（怎么工作、在工艺里干嘛）→ **轨1**（到理解流程的深度）。
- 产业 / 价值（谁做、多大、壁垒、国产化、价值量）→ **轨2**；材料的**深度技术原理**作为"为什么难 = 壁垒"的铺垫，**跟轨2赛道页走**。
- 判据：**"为看懂工艺" → 轨1；"为看懂生意 / 壁垒" → 轨2。**

**回补接口**：轨2 研究中发现"看懂工艺本该有、轨1却没讲"的点，**别打断轨2**——登记到 `TRACK1-BACKFILL.md`（一句话一行），攒一批后任意 session 集中回补 `principles/`（`chip-explainer.html`）：每补一项 → 真机渲染核对 → 发布 → 更新状态。该文件是两轨的**异步交流队列**，已预置初始缺口（光刻胶机制、CMP 工序、电子特气角色）。

## 5. 单赛道页模板（可复用，先用材料跑通）
每个赛道页统一八段：
1. 一句话定位 + 在产业链的位置（小图/高亮）。
2. 细分品类拆解（表/卡）：每类 → 价值量/份额、关键玩家、国产化进度。
3. 价值量与占比（图）：该环节占晶圆厂支出 / 整链价值的比例。
4. 护城河（为什么难做、为什么换不掉）：纯度 / 资质认证周期 / 客户绑定 / IP / 规模。
5. 玩家格局：国际龙头 + 国产玩家 + 份额（标年份/来源）。
6. 驱动与催化：晶圆增长、先进制程提升材料强度、国产替代、地缘。
7. 风险。
8. （可选）可刷新数据点。

## 6. 从「材料」开始（含补充建议）
材料是好的切入点：**承接第一轨的硅片·掩膜**，且国产替代叙事强。两点补充：
1. **先搭轻量「价值地图总览」骨架**（`value-chain/index.html`），把 设计→设备→材料→制造→封测 各环节与相对价值/壁垒先摆出来，让材料落进整张图（骨架 + 材料深度页即可，不必一次做满）。
2. **单赛道做成可复用模板**（见 §5），材料跑通后设备/EDA/存储/封装照搬，结构一致、可横向比较。

材料赛道细分（逐个填模板）：
- 硅片（大硅片：信越 / SUMCO / 环球晶 / 中环 / 沪硅）
- 光刻胶（JSR / 东京应化 / 信越 / 杜邦；EUV 胶高壁垒；南大光电 / 彤程新材）
- 电子特气（林德 / 空气化工 / 液空；华特气体 / 金宏）
- CMP 抛光液 & 抛光垫（卡博特 / 慧瓦 / 陶氏；安集科技 / 鼎龙）
- 靶材（JX / 霍尼韦尔；江丰 / 有研）
- 掩膜版 / 掩膜基板 blank（Hoya / AGC / 信越；清溢 / 路维）
- 前驱体、湿电子化学品、封装材料（基板 / 塑封料 / 键合丝）等
- 主线：**越先进的制程，材料种类更多、纯度更高、价值量更大**；以及**国产化率与卡点**。
