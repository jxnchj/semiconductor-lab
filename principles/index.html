先理清层级：晶圆 ≠ 芯片 ≠ 晶体管（这三者是"大套小"的包含关系）
一句话记住：晶圆是一大片硅原料 → 切成几百颗芯片(裸晶) → 每颗芯片里又有几百亿个晶体管。它们是层层包含、尺寸差着十万倍的三样东西，绝不是一回事。
尺度跨度：放大三级（封装 → 裸晶 → 晶体管阵列）

三张图常被误解，逐一澄清：
① 左图那些金色圆点是芯片封装底部的焊球（或引脚），不是硅片本身——芯片靠它们焊到电路板、与外界通电通信。
② 中图的一颗裸晶（Die）就是"一整块芯片"：一片指甲盖大的硅，内部按功能划成多个区——通常有好几个 CPU 核、一个 GPU、大块缓存、I/O 接口等（不是只有一个核）。
③ 右图是放大后的晶体管阵列：青色横条是"有源区"（源极/漏极所在），铜色竖线是一条条"栅极"。每一个铜线与青条的交叉点，就是一个晶体管（颜色与 晶体管页完全对应）。整片硅上这样的交叉点有数百亿个。
关键澄清：晶体管是"长"出来的，不是"装"上去的
很多人以为芯片是"一块硅板上面镶了很多晶体管"——这是把电路板的直觉套错了。真相是：晶体管不是单独做好再贴上去的零件，而是用光刻一层层在硅本身上"雕"出来的——掺杂出来的区域就是源极 / 漏极，中间没掺杂的硅就是沟道。所以一颗芯片是一整块不可分割的硅（单片集成），里面拆不出"一个独立的晶体管"。而且是先在整片晶圆上把成百上千颗芯片同时造好，最后才切开，不是一颗一颗装出来的。
基础名词（每个都配了示意图）
半导体 Semiconductor

导电能力介于导体与绝缘体之间、可以被控制的材料（主要是硅）。正因为"可控"，才能拿它做开关。

晶圆 Wafer

一大片圆形的纯硅薄片（常见直径 200/300mm），是制造芯片的原材料。一片能造出几百颗芯片。

裸晶 / 芯片 Die

从晶圆上切下来的一小块，就是一颗芯片的全部电路；内部分成 CPU 核、缓存等功能区。

封装 Package

给裸晶加上外壳和引脚/焊球，保护它、并能焊到电路板上。我们平时见到的"一颗芯片"就是封装后的样子。

晶体管 Transistor

芯片里最基本的"电控开关"，一颗芯片上有几百亿个。无数开关组合起来就能运算、存储。

栅 / 源 / 漏 / 沟道

晶体管的几个部位：栅极是控制端，源极、漏极是电流两端，中间的硅是沟道（电流的通路）。

掺杂 Doping

往纯硅里掺入微量杂质（金色），改变它的导电性，从而造出源极、漏极这些导电区域。

光刻 Lithography

用光透过模板把电路图案"印"到硅片上，是芯片制造里最关键、最精密的一步。

制程节点 Node

"Xnm"那个代号，代表一代制造工艺（数字越小越先进）。注意：如今它已不是真实的物理尺寸。

摩尔定律 Moore's Law

经验规律：同样面积上的晶体管数，大约每两年翻一倍。芯片几十年来越做越强，背后就是它。

术语图鉴 · 速查（接着基础名词，按类别各配示意图）
材料与上游（硅片·掩膜）
晶圆 / 硅片

镜面单晶硅圆片，造芯片的"画布"；左下小缺口标晶向，主流 300mm。

单晶硅 / 多晶硅

原子整齐排成一片=单晶(造芯片)；许多小晶粒=多晶硅(由改良西门子法提纯，是原料)。

直拉法 CZ

籽晶接触熔硅、边转边极慢上提，拉出整根单晶硅锭——主流长晶法。

区熔法 FZ

线圈把一小段硅熔成"浮区"慢慢扫过，不碰坩埚→更纯，多用于功率器件衬底。

晶向 <100>

单晶里原子排列的方向，影响后续工艺与解理；芯片主流用 <100> 面。

电子级 / 太阳能级

造芯片要电子级 11 个 9(99.999999999%)；光伏用太阳能级 6~9 个 9——两个赛道。

pellicle 保护膜

掩膜上方的防尘薄膜：灰尘落在膜上(离焦)而非掩膜面，免得被印到每颗芯片。

器件与结构（晶体管 / 先进结构演进）
栅氧化层

栅极与沟道之间极薄的绝缘膜，靠它"隔空"控制沟道。

反型层

加栅压后被"现拉"到硅表面的电子薄层，就是导通时的沟道。

MOSFET（场效应）

用栅极电场隔空开关源漏间电流的晶体管，逻辑芯片的主力。

CMOS（互补对）

NMOS+PMOS 一对互补开关，搭出逻辑门、低功耗，数字芯片的基础。

FinFET（鳍式）

沟道立成一片"鳍"，栅极从三面包住，比平面控制力强（14/7nm 主力）。

GAA 纳米片

沟道做成几层纳米薄片，栅极 360° 环绕每片，3nm/2nm 方向（Intel 称 RibbonFET、三星称 MBCFET）。

短沟道效应

沟道太短时栅极管不住、漏电——逼出 FinFET / GAA 这些新结构。

CFET（互补 FET）

把 NMOS 与 PMOS 上下叠起来，再省一半占地——GAA 之后的下一步结构。

制造与光刻（制造流程 / 光刻）
掩膜（光罩）

刻有电路图案的母版，光透过它把图案印到硅片上。EUV 用反射式。

光刻胶

涂在硅片上的感光材料，曝光后显影出图形；EUV 胶是高壁垒耗材。

DUV / EUV

193nm / 13.5nm 曝光光源；波长越短线越细。EUV 全球仅 ASML 能造。

NA / High-NA

镜头数值孔径，收光锥角越大线越细；0.33→0.55 即 High-NA EUV。

多重曝光 SADP

一次画不够密，就用侧墙自对准把线翻倍——DUV 做 7nm 全靠它。

CMP 抛光

化学机械抛光：把上一步沉积出的凹凸表面磨平，才能继续往上叠。

沉积 / 刻蚀

加料（沉积一层薄膜）与去料（刻蚀挖掉）——制造里反复交替的两招。

退火

高温激活掺入的杂质、修复晶格损伤，让源漏真正"成形"。

STI 隔离

器件之间刻出浅沟槽填氧化物，把相邻晶体管彼此绝缘隔开。

侧墙 / LDD

栅两侧的弧形介质侧墙＋浅而轻的掺杂延伸，抑制短沟道漏电。

硅化物

在源/漏/栅顶自对准生成金属硅化物（钢灰），大幅降低接触电阻。

大马士革 / via·ILD

嵌铜布线：在层间介质 ILD 里刻槽填铜，竖直的通孔 via 连上下层。

制程与经济（制程节点与经济）
密度 MTr/mm²

每平方毫米晶体管数，真正可比的先进度指标（"nm"已非物理尺寸）。

FEOL / BEOL

前段造出1 层晶体管（只一遍）、后段叠十几层金属布线（反复做）。

良率 Yield

一片晶圆里好芯片的比例；die 越大越易碰到缺陷、良率越低、成本越高。

功耗墙

主频 ~2005 后停涨，受功耗 / 发热限制（Dennard 缩放终结）→ 转多核。

背面供电（PowerVia/BSPDN）

把供电网络挪到晶圆背面，正面全腾给信号线 → 降压损、提性能与密度。

存储器
DRAM（1T1C）

1 晶体管+1 电容存 1 bit，会漏需刷新（做主存/显存）。三星/海力士/美光。

SRAM（6T）

6 管锁存，快、不刷新但占地大（做片上缓存 Cache）。

3D NAND

上百层字线竖叠+垂直沟道，靠"加层数"降成本（非易失，SSD）。

HBM

DRAM 垂直叠+TSV 的高带宽显存，AI 标配（详见 先进封装）。

存储层级

寄存器→SRAM→DRAM→NAND→HDD，越上越快越贵越小。

封装与集成（先进封装）
2.5D / 3D

并排在中介层上（2.5D）/ 垂直堆叠、TSV 直连（3D）——两条集成路线。

CoWoS

台积电 2.5D 封装：GPU 与 HBM 并排放在硅中介层上，AI 芯片标配。

中介层 Interposer

带 TSV + 精细布线(RDL) 的硅板，把并排芯片用上万根超短走线连起来。

TSV 硅通孔

竖直穿透芯片的铜柱，让信号 / 电源上下直连，是 3D 堆叠的关键。

凸点 BGA/C4/微凸点

由粗到细的三级焊点：BGA(接主板)→C4(接中介层)→微凸点(接芯片)。

Chiplet

把大芯片拆成多颗小芯粒分别造、再用封装拼起来（UCIe 互连）。

混合键合

铜垫直接对接+介质键合，无焊料、亚微米间距，更密更省电（HBM4/3D）。

功率 / 化合物
SiC / GaN（宽禁带）

禁带更宽 → 耐高压高温、低损耗、高频；用于电动车 / 光伏 / 快充 / 射频。

其它（纯概念 / 产业角色）
单片集成 / 易失·非易失
晶体管在硅本身"雕"出、整块不可分；断电丢(SRAM/DRAM) vs 断电留(NAND)。
binning / Dennard 缩放
按性能·缺陷分档出货（满血/屏蔽版）；缩小同时保持功耗密度（约 2005 终结）。
EDA / IP
芯片设计软件（Synopsys / Cadence / 西门子）/ 可复用电路模块（如 ARM）。
代工 / IDM / OSAT / WFE
只代工(台积电) / 设计制造一体(Intel·三星) / 封测厂(日月光·长电) / 晶圆制造设备(ASML·AMAT·Lam·TEL·KLA)。
SoC / NPU
SoC＝把 CPU/GPU/NPU/IO 等整合进一颗芯片（片上系统）；NPU＝专做 AI 矩阵运算的处理器。
VDD / GND
芯片的电源(VDD) 与地(GND)，是数字电路的两条供电基准。
LPP（EUV 光源）
EUV 的产光方式——CO₂ 激光打锡滴产生 13.5nm 光（详见 光刻）。
IGBT
绝缘栅双极晶体管，高压大电流的功率开关（电车/工业逆变，详见 功率/化合物）。

二、掩膜版：把电路图变成光刻的"底片"

反射，做不了透射式
关键事实：掩膜是 4× 放大版（投影时缩小 4 倍，缓解掩膜制造精度压力）｜ 一套先进芯片要 几十~上百张掩膜（每层一张）｜ 一张高端 EUV 掩膜可达数百万美元 ｜ 表面贴 pellicle 薄膜挡灰（落在掩膜上的尘点会被一并印到每颗芯片上）。掩膜用在光刻——详见 光刻。

晶体管 = 一个电控开关。源极 (S) 和 漏极 (D) 是电流的两端，中间那段硅是沟道。上方金属是 栅极 (Gate)，与沟道之间隔着一层极薄的 栅氧化层（绝缘层）——栅极不碰沟道，只靠"电场"隔空控制它。立体图默认半透明，能看清栅极如何"骑"在沟道之上、又被栅氧化层隔开（上方 ▭/⊓/▤ 可切换平面/FinFET/GAA 纳米片三种结构：栅极从盖 1 面 → 包 3 面 → 360° 环绕每片沟道，越包越紧、对短沟道漏电的控制力越强）；栅极沿电流方向的长度就是"栅极长度 Lg"，正是历史上定义"制程"的那个尺寸。
点⚡加电压：电场把电子吸到沟道表面形成通路。注意方向——电子 e⁻ 从源极流向漏极（青点动画），而习惯上说的"电流 I"方向相反，从漏极流入、源极流出。
那"沟道"本身又是怎么凭空出现的？接着看下面的反型层 5 步交互。
沟道是怎么"拉"出来的？——栅极把"反型层"现场造出来（点"下一步"看 5 步）
空穴 (+)：P 硅多数载流子 电子 (−)：来自源/漏 N⁺ 耗尽区 电场 E
这就是"场效应"的核心：沟道不是预先埋好的导线，而是栅压用电场"现拉"出来的反型层。撤掉电压它立刻消失——所以晶体管能"关断"、能当开关。决定通不通的不是"掺没掺杂"，而是栅压有没有在表面拉出自由电子：横向只有栅极正下方有电场，纵向电子被电场按在栅氧界面那一两纳米、下方靠耗尽区与体隔开；加上源 N⁺→体 P→漏 N⁺ 是背靠背 PN 结，唯一能连续导电的通路就是这层表面反型沟道。
从开关到运算：CMOS（一对互补开关）
单个晶体管只是开关，怎么就会"算"？关键是把它们搭成逻辑门。最基本的 CMOS 用一对互补的开关——上面的 PMOS 负责把输出拉到 1，下面的 NMOS 负责把输出拉到 0（注意：前面 3D/2D 只画了 NMOS 一种，真实逻辑里两种成对出现）。图中就是个反相器：输入 0 出 1、输入 1 出 0。再用反相器搭出与非 / 或非等门，门又能拼出加法器、锁存器……上亿个门连起来，就是"运算与存储"。CMOS 的妙处是永远一开一关、几乎只在翻转时耗电，所以能做到低功耗、被全行业采用。（这条线再深入就是逻辑 / 电路设计与 EDA，本讲解器只点到为止。）
一颗芯片有多少层晶体管？
在主流逻辑芯片里，晶体管只有最底下薄薄一层（铺在硅表面）。它们上方再叠 10~20 层金属布线（称 BEOL 后段工艺），像立交桥一样把上亿个晶体管按电路连起来——所以芯片"很多层"，但那些大多是"布线层"，不是"晶体管层"。真正的多层晶体管堆叠（3D），见 先进结构演进（3D 那段）。
为什么要这么多层布线？单个晶体管只是一个开关，自己不会"算"。布线的作用是把上亿个开关连成电路：让一个晶体管的输出（漏极）去控制另一个的输入（栅极），再接上电源、地、时钟和对外引脚——信号（0 / 1）在这张开关网络里层层触发、流动，就是"运算"。叠这么多层，是因为要连的线实在太多（布线密度）：下层细而短走本地连接，上层粗而长走全局信号与供电——不是"一层对应一个晶体管"。
整条芯片生产分大三段：前道（晶圆制造） → 中测（CP） → 后道（封装测试）。这一页就是核心的「前道」——在晶圆上把电路一层层造出来，与「后道·封装测试」前后对应。

这一页讲"制程节点"到底是什么：先搞清"nm"这个名字今天还剩多少真实含义，再看制程推进绕不开的两道约束——为什么越先进越贵（良率与成本）、为什么不能无限快（功耗墙）。至于"不能无限小"该怎么破（改变晶体管结构），见「先进结构演进」。
相对密度 ~1×
"制程节点"是什么？ 早期它确实指晶体管的关键尺寸（如栅极长度）：28nm 比 14nm 大一倍，线条更粗。每缩小一代，单位面积晶体管数大约翻倍多，于是更快、更省电、单颗更便宜——这就是"摩尔定律"。但从约 2009 年起，"nm"这个数字就不再等于任何真实物理尺寸了。今天它表示的是相对上一代的密度、功耗、性能，更像一个"性能档位标签"。
三大厂的命名标准（互不统一，同名≠等价）
台积电 TSMC
N7 → N5 → N3 → N2 → A16/A14。N3 仍用 FinFET，N2 起改用 GAA。"A"=埃(0.1nm)，进入"埃米时代"。全球约九成 5nm 以下芯片由它代工。
三星 Samsung
改用 SF 体系：7LPP(首批量产 EUV) → 5LPE → 4LPE → SF3(3nm，业界最早上 GAA，自称 MBCFET) → SF2 → SF1.4。
英特尔 Intel
曾用"真实"nm，但其 10nm 密度≈别家 7nm，数字上显得落后。2021 改名：Intel 7 → 4 → 3 → 20A → 18A。20A/18A 首发 RibbonFET(GAA) + PowerVia(背面供电)，密度上对标台积电 N2。
没有行业统一标准，真正可比的是密度（每平方毫米晶体管数 MTr/mm²）、功耗与性能。举个反差例子：台积电 N3E 的最小金属间距约 23nm、栅极间距（CPP）约 45–48nm，密度约 170–200 MTr/mm² 量级——"3nm"这个名字里，根本找不到 3nm 这个尺寸。
为什么越先进越贵：良率与成本
晶圆上的缺陷是随机分布的，一个缺陷基本会让所在的整颗 die 报废。die 越大，单颗碰到缺陷的概率越高、良率越低——同样 5 个缺陷，大 die 坏掉 5/16，小芯粒只坏 5/64。叠加先进制程本身更贵（EUV、多重曝光、更多道工序）和大面积难做，先进大芯片成本飙升。这正是 Chiplet（把大芯片拆成小芯粒）的根本经济动机；卖的时候还会按性能 / 缺陷分档（binning）——满血版、屏蔽版分级出货。
为什么不能无限快：功耗墙
晶体管还在变多（摩尔定律仍在走），但主频从 ~2005 年起就基本停涨了。原因是 Dennard 缩放终结：以前晶体管缩小，单位面积功耗能保持不变；后来漏电（静态功耗）急剧上升，再加上动态功耗 ∝ 频率×电压²，继续拉高频率会让功耗与发热失控——这堵墙就叫"功耗墙"。于是行业转向多核并行 + 专用加速（GPU / NPU / DSA）、以及靠先进封装堆更多算力。对芯片来说，今天的硬约束往往不是"能不能更快"，而是"功耗 / 散热允不允许"——这也是 AI 数据中心拼液冷、供电、能效的根因。
前一页「制程节点与经济」讲了制程的"标尺"与经济 / 功耗约束。这一页讲怎么在结构上继续推进：先看为什么不能无限缩小（短沟道漏电），再看晶体管怎么把栅极包得更紧（FinFET→GAA→CFET）、把供电搬到背面，并走向 3D 堆叠。（光刻 EUV / High-NA 见独立的"光刻"页）
A · 为什么不能无限缩小：短沟道漏电
解决办法不是只把栅极做短，而是改变晶体管的形状，让栅极从更多方向把沟道"包住"，重新夺回控制力 ↓
B · 晶体管结构进化：把栅极包得更紧（栅氧化层始终都在）
沟道（硅本体）源/漏（掺杂）栅氧化层(薄)栅极→ 电流：源→漏，沿沟道流

平面 Planar（旧）

栅极只盖在沟道顶部 1 面。中间那条黄色就是栅氧化层（一直都有，只是很薄）。缩小后栅极管不住 → 漏电。

FinFET（鳍式·立体）

沟道立成一片鳍（蓝），两端是源/漏（绿）。栅极（橙）隔着黄色栅氧化层从3 个面包住沟道，控制力大增。14/10/7nm 主力。

GAA / 环绕栅（立体）

沟道做成几根纳米薄片（蓝），每根都裹一圈栅氧化层（黄），栅极（橙）360° 环绕。控制力最强，3nm/2nm 方向。

三种结构里栅氧化层从未消失——它始终是栅极与沟道之间那层薄绝缘膜，只是从"盖一面"变成"包三面""绕四面"。电流都一样：沿沟道从源极流到漏极，由栅极在中间隔空开关。
C · CFET：GAA 之后，把互补的两种管子上下叠
CFET（互补 FET）把互补的一对（NMOS + PMOS）上下叠起来，省掉并排的横向面积 → 同样功能占地约减半，是 GAA 之后（"埃米时代"）的延续方向。目前还在研发 / 试产（imec、Intel、台积电已在 IEDM 展示测试结构），量产更靠后（约 2030 前后）。
D · 背面供电（PowerVia / BSPDN）：把供电网络挪到芯片背面
传统芯片信号线和供电线都挤在正面的十几层金属里，互相抢空间、供电路径也长。背面供电（BSPDN）把供电网络搬到晶圆背面，用纳米级 TSV 从背面直达晶体管：正面腾出来专走信号、布线更松，供电路径更短 → 压降更小、性能 / 能效更好。进展：Intel 18A（PowerVia）已量产，台积电 A16（超级电轨 SPR）2026 下半年，三星 SF2Z（2027）跟进——是 2nm 世代竞争的关键变量。
E · 芯片是平面一层，还是多层晶体管？
回答你的问题：绝大多数逻辑芯片（CPU/GPU）的晶体管确实只有平铺的 1 层，靠上面的多层金属布线连接。但"3D"已经无处不在：① 3D NAND 闪存早已把存储单元垂直堆叠 100~300+ 层；② 先进封装 / Chiplet 把多颗裸晶上下键合（如 HBM 显存、AMD 的 3D V-Cache）；③ 逻辑上正在研发 CFET（把两种晶体管上下叠）和背面供电。趋势就是从"平面缩小"转向"向上叠高"。
附：另一条路线之争 —— 华为「韬(τ)定律」
📌 提醒：这是"厂商提案 / 战略叙事"，不是摩尔定律那样被业界长期验证的"定律"，读时请打个折。
2026 年 5 月，华为在 IEEE ISCAS 2026（上海）由何庭波提出「韬(τ)定律」(Tau Scaling Law)：与其继续几何缩微（把晶体管越做越小、依赖 EUV），不如转向"时间缩微"——系统性压低电路的时间常数 τ（信号翻转一次所需的基本时间），配合 LogicFolding（逻辑折叠）等设计手段，在不依赖 EUV 的前提下提升等效性能与密度。华为称该思路过去六年已用于约 381 款量产芯片，2026 秋首批麒麟芯片将采用 LogicFolding，并称高端芯片密度有望 2031 年达到等效 1.4nm；这也被称为中国企业首次在全球半导体领域提出体系化的演进范式。
怎么看（投研视角）：本质是对先进光刻设备受限的公开回应——把竞争从"造更小的管子"转向"用设计 / 架构换性能"。方向上与本页的 背面供电、3D 堆叠、Chiplet 同属"不靠单纯缩微"的延续；但其性能 / 密度承诺多为华为自述、面向未来，需第三方数据验证，"定律"二字更多是叙事。作为非 EUV 路线的代表性思路，对理解中国半导体的突围方向有参考价值。
光刻：芯片制造的"画笔",也是最大的卡脖子环节
光刻 = 用光透过掩膜（光罩）把电路图案印到涂了光刻胶的硅片上（这套动作就是下面这套"图案转印"）。能画多细由瑞利公式决定：分辨率 ≈ k₁ · λ / NA——波长 λ 越短、镜头数值孔径 NA 越大，线就越细。于是先进制程把光源从 DUV（193nm）一路推到 EUV（13.5nm），并不断加大 NA。
这套动作叫「图案转印」：涂胶 → 曝光 → 显影 → 刻蚀 → 去胶，制造里每做一层新图形就重复一次
① EUV 的光怎么"造"出来：锡等离子体光源（LPP）
EUV 不是一盏"灯"——它是用大功率 CO₂ 激光每秒轰击约 5 万颗锡(Sn)液滴，把锡打成等离子体，等离子体辐射出 13.5nm 极紫外光；再用椭球收集镜聚到"中间焦点"送进光学系统。这条链路效率极低（注入功率 → 可用 EUV 仅约百分之几），整机功耗兆瓦级，且必须全程真空。光源的稳定与功率，正是 EUV 良率与产能的命门。
② 光怎么"走"：DUV 折射式 vs EUV 反射式（全程真空）
一句话对比：DUV（193nm）＝折射式 —— 用玻璃透镜+透明掩膜（光穿过），可在空气中、或浸没在水里（n≈1.44，把 NA 抬到 1.35）；EUV（13.5nm）＝反射式 —— 会被玻璃和空气吸收，所以全改用镜子、全程真空、连掩膜都反射。下面分别画出两者光路：
这是 EUV 最反直觉、也最贵的地方：13.5nm EUV 会被几乎所有材料（玻璃、空气）吸收，没法像 DUV 那样用透镜。于是整条光路全部改用反射镜，连掩膜都是反射式的（图案做在镜面上，斜约 6° 入射）。镜子是钼/硅(Mo/Si)多层膜反射镜，单面也只反射约 70%；一台机器十多面镜子叠下来，光走到硅片只剩百分之几——这就是为什么光源功率要拼命做大。掩膜还要加一层极薄的防护膜 Pellicle 挡颗粒，本身就是难做的高价耗材。
③ 怎么画得更细：两个杠杆——更大的 NA，和多重曝光
NA（数值孔径）= 镜头/镜子能收集光的锥角;NA 越大，光能聚成的点越小、线越细（分辨率 ≈ k₁·λ/NA）。把 NA 从 0.33 提到 0.55 就是 High-NA EUV，单次曝光从约 13nm 提到约 8nm。代价：它用变形(anamorphic)光学（一轴 8×、另一轴 4× 缩小），曝光视场减半，大芯片要拼接(stitching)。
多重曝光：单次分辨率不够时，把一层图形拆成多块掩膜 / 多道工序叠出来。图中 SADP 自对准双重成形——先做较稀的"芯轴"，侧壁长一圈侧墙，去掉芯轴后侧墙变成两倍密的线。DUV(193nm) 时代正是靠它硬做到 7nm；EUV 能少拆几次，但最细处仍需多重曝光。每多一次 = 多一套掩膜、多一道工序 = 更贵更慢，这是先进制程成本飙升的重要原因。
为什么光刻是最硬的卡脖子环节：EUV 光刻机全球只有 ASML 一家能造，一台标准 EUV ~1.5 亿美元、High-NA（EXE:5200B）约 3.8 亿美元，年产量有限。配套同样高度垄断：光刻胶（JSR / 东京应化 / 信越 / 住友，EUV 金属氧化物胶 Inpria）、掩膜基板（Hoya / AGC）、防护膜 Pellicle（三井）、光学（蔡司）。进展（2026）：Intel 率先用上 High-NA（EXE:5200B，175 片/小时），三星、SK 海力士跟进；台积电因太贵暂缓（A14 暂不用 High-NA），High-NA 量产普遍预期 2027–28。对国产替代，EUV 仍是尚未突破的最高壁垒。
芯片分两大类：逻辑(负责运算)和存储(负责记忆)。存储又分易失(断电就丢：SRAM / DRAM)和非易失(断电仍在：NAND / NOR 闪存)。存储是强周期、寡头的大产业，也是 AI 最关键的"喂数据"瓶颈。
A · 存储金字塔：越往上越快越贵越小，越往下越慢越便宜越大
同一台电脑里有一整条存储层级：越靠近 CPU 越快但越贵越小（寄存器 / SRAM 缓存），越往外越慢但越便宜越大（DRAM 内存 → NAND 固态盘 → 机械硬盘）。其中真正属于半导体存储芯片的是 SRAM、DRAM、NAND 三类——下面逐一看它们的"存一个比特"长什么样、为什么难缩小。
B · DRAM（动态内存）：1 个晶体管 + 1 个电容 = 1 bit
DRAM 用 1 个晶体管 + 1 个电容（1T1C）存一个比特：电容里有没有电荷代表 1 或 0，晶体管是访问开关（栅极连"字线 WL"，一端连"位线 BL"）。麻烦在于电荷会慢慢漏掉，必须每隔几十毫秒刷新一次（"动态"由此得名）；而电容要够大才存得住电荷，于是越缩越难——只能做成极高深宽比的深沟 / 柱状结构。DRAM 微缩逼近极限，正是 HBM 用"垂直堆叠"换容量与带宽的根本原因。主要玩家：三星 / SK 海力士 / 美光，国产 长鑫 CXMT。
C · SRAM（静态缓存）：6 个晶体管锁住 1 bit——快，但占地大
SRAM 用 6 个晶体管把一个比特"锁"住：两个反相器首尾相接成双稳态锁存器（保持 1 或 0），外加两个访问管接到位线。好处是极快、通电即保持、不用刷新，所以用作 CPU 片上缓存 Cache。代价是6 管占地大、且越来越难缩小（先进节点上 SRAM 面积几乎不再缩）——这正是高端芯片缓存容量金贵、并催生 3D V-Cache（把 SRAM 叠到逻辑上，AMD）的原因。SRAM 通常不单独成厂，集成在逻辑芯片里，由代工厂 / 设计公司一起做。
D · 3D NAND（闪存）：平面塞不下了，就把存储单元"竖着叠"
NAND 闪存把电荷存在每个单元的电荷俘获层里（断电也不丢 = 非易失）。平面 NAND 缩到极限后，改走 3D：把上百层"字线 / 控制栅"竖着叠起来，再用一道刻蚀打出垂直沟道孔穿过所有层——每个"字线 × 沟道"的交点就是一个存储单元，一根柱子上的单元串成一串(NAND string)。妙处是靠"加层数"而不是"缩尺寸"来降低每比特成本，所以层数竞赛(200→300→400+)成了主线。玩家：三星 / SK 海力士(含 Solidigm) / 美光 / 铠侠·闪迪，国产 长江存储 YMTC——也是国产存储突破最快的方向。
从研究角度记住三点：① 存储是典型的强周期、寡头生意——产品高度标准化(近似大宗商品)，供需错配→价格大起大落，景气周期对相关公司业绩与股价影响极大。② 格局：DRAM ≈ 三星 / SK 海力士 / 美光(+长鑫)，NAND ≈ 三星 / SK 海力士 / 美光 / 铠侠·闪迪(+长江存储)。③ HBM 改变了游戏：它把本属周期品的 DRAM 变成绑定 AI、供不应求的高毛利产品(见 先进封装)，是当前存储里最贵、最紧的细分。

终测 FT 在不同温度/电压下全功能测试、分档（binning）出货。 中测 CP＝在"整片晶圆"上测（挑掉坏的，不浪费封装钱）；终测 FT＝在"封装好的成品"上测。两道测试一前一后。
键合两条路线：引线键合 vs 倒装（Flip-Chip）
封装外形也在不断进化：DIP → QFP → QFN → BGA（底部焊球阵列）→ 晶圆级 WLP / Fan-out → 2.5D / 3D 先进封装（CoWoS / HBM / Chiplet）。摩尔定律放缓后，先进封装成了继续提性能的主路——详见 先进封装。
谁在做（接第二轨）：后道由封测厂 OSAT 承担——日月光（ASE）、安靠（Amkor）、长电科技、通富微电、华天等。它们的产业格局、价值量、国产化属第二轨的封测赛道，本页只讲"工艺怎么走"。
摩尔定律放缓后，"把多颗芯片拼起来 / 叠起来"成了继续提升性能的主路——这就是先进封装。今天最典型、也最紧缺的应用，就是 AI 加速器：一颗 GPU 主芯片 + 一圈 HBM 高带宽显存。
A · 思路转变：从"单片大芯片"到"芯粒(Chiplet)拆分 + 封装拼装"
把一颗巨大的芯片拆成几颗小"芯粒"(chiplet)分别制造、再用封装拼成一个整体，好处是：① 绕开光罩面积上限（单次曝光最多约 858mm²）；② 小芯片良率更高、更省成本；③ 可以混搭不同制程（算力核用最先进的，IO 用成熟的）。代价是芯片之间要靠先进封装高速连起来——这正是下面要讲的。
B · 两条路线：2.5D（并排）vs 3D（垂直叠）
硅（芯片 / 中介层 die） 铜（TSV / 布线 / 微凸点） 焊料球（C4 / BGA） 封装基板
一句话分清：2.5D 是把芯片"摊开"排在一块硅中介层上（靠中介层里的精细布线横向高速连）；3D 是把芯片"叠起来"（靠 TSV 硅通孔竖直打穿、上下直连，路径更短、更省电）。真实的 AI 芯片往往两者并用：HBM 自己是 3D 叠的，再和 GPU 一起 2.5D 放到中介层上。下面分别拆开看。
C · CoWoS 精细剖面：一颗 AI GPU 是怎么"拼"出来的（从下往上看）
从下往上：主板 PCB → BGA 焊球 → 封装基板 → C4 凸点(~100µm) → 硅中介层 → 微凸点(~10µm) → GPU + 多颗 HBM。中介层是关键——它是一小片硅，上面有极精细的布线(RDL)和竖直的 TSV 硅通孔，把并排的 GPU 与 HBM 用上万根超短走线连起来，带宽远超普通基板。越往上凸点越小越密（基板用大焊球、中介层用微凸点），逐级把"芯片级"的细间距过渡到"电路板级"的粗间距。注：NVIDIA Blackwell 就是把两颗 GPU die 用 CoWoS-L（内嵌硅桥）拼成"一颗"，以突破单次光刻的面积上限。
D · 放大一颗 HBM：垂直叠 8~16 层 DRAM，用 TSV 打竖井
HBM（高带宽存储）把一颗基底逻辑 die + 上面叠 8~16 层减薄的 DRAM用 TSV 硅通孔竖直打穿、上下直连，做成一个"存储立方体"。因为连接是又宽又短的竖井，带宽极高、功耗低。当下主力 HBM3E：8/12 层、单颗 24/36 GB、带宽约 1.2 TB/s（一颗 AI GPU 旁边放 6~8 颗，如 NVIDIA B200＝8 颗 ≈ 192 GB）。2026 量产的 HBM4：接口位宽翻倍到 2048-bit、带宽约 2 TB/s 以上、最高 16 层 / 48 GB，并改用混合键合（见下）。
E · 层间互连的进化：微凸点 → 混合键合
为什么要换？传统微凸点是"铜柱顶个焊料球"把上下两片芯片接起来，中间有缝、间距停在 ~10µm 量级；当堆叠层数越来越多、I/O 越来越密，焊料的电阻、热阻和间距都成了瓶颈。混合键合直接把两片芯片的铜垫对铜垫压合、周围介质也键合在一起，没有焊料、没有缝隙，间距能进到亚微米——更密、更省电、散热更好。HBM4 与台积电 SoIC（如 NVIDIA Rubin、AMD 3D V-Cache）正是这条路。
F · 产业现状：为什么先进封装＝当前 AI 算力的瓶颈
2.5D 封装
CoWoS 几乎由台积电独家大规模供应；2026 年 NVIDIA 一家就订走约六成产能，交期一度排到 50+ 周。台积电正把 CoWoS 产能向 ~13 万片/月扩。
HBM
三家：SK 海力士（领先）/ 三星 / 美光。HBM3E 当前主力，HBM4 于 2026 量产（SK 海力士 16 层 48GB、美光 12 层 36GB、三星 2026 初出货）。
谁在用
NVIDIA Blackwell（双 GPU die＋8×HBM3E）、下一代 Rubin（HBM4＋CoWoS-L＋SoIC 3D 堆叠）；AMD MI300/MI350 系列。先进封装＋HBM 已是 AI 算力扩张最紧的两个卡点之一。
一句话收尾：当晶体管已经难再快速变小，"把更多芯片更密地拼起来、叠起来、用 HBM 喂饱它们"成了 AI 芯片继续变强的主战场——所以先进封装(CoWoS) 与 HBM 的产能，直接决定了能造出多少 AI 算力。
除了"硅基逻辑 / 存储"这条主线，还有一条独立赛道：功率半导体 + 化合物半导体（SiC / GaN）。它们不拼"更小更快"，而拼扛高压、耐高温、低损耗、高频——是电动车 / 光伏储能 / 快充 / 5G / 电网的核心器件，也是和 AI 主线并行、国产化更快的投资方向。
A · 关键在"宽禁带"：Si vs SiC vs GaN
普通芯片用的是硅(Si)。而 SiC(碳化硅)、GaN(氮化镓) 是"宽禁带"半导体——禁带宽度约为硅的 3 倍。宽禁带意味着：能承受更高电压、更高温度，开关更快、能量损耗更低。所以它们不去和先进逻辑拼纳米，而是专攻功率转换(把电"变压/变流")和射频：同样一台电动车逆变器、一个快充头，用 SiC/GaN 能做得更小、更凉、更省电。
B · 用在哪：SiC 管高压、GaN 管高频

硅 Si（传统）

中低压的消费 / 工业 / 家电，量大、便宜、最成熟。今天绝大多数功率器件仍是硅（IGBT / MOSFET）。

GaN 氮化镓（高频·中低压）

约 100–650V：快充 / 电源适配器、数据中心供电、射频（5G 基站 / 手机）。优势是开关极快、体积小。

SiC 碳化硅（高压·大功率）

600–1700V+：电动车主驱逆变器、光伏 / 储能、充电桩、电网。优势是耐高压高温、省电、续航更长。

C · 产业链与主要玩家
器件类型
功率 MOSFET / IGBT / 二极管——做开关与整流，把电"变压、变流"（逆变 / 整流 / DC-DC）。
SiC 产业链
衬底（长晶最难＝最大壁垒）→ 外延 → 器件 / 模块；车规可靠性门槛高。衬底是价值与壁垒最集中处。
国际玩家
英飞凌 Infineon、意法 ST、Wolfspeed、安森美 onsemi、罗姆 ROHM。
国产玩家
三安光电、时代电气、斯达半导、天岳先进（衬底）、天科合达 等——国产化进展相对较快。
从研究角度：这条赛道与先进逻辑的卡脖子点不同——壁垒在衬底长晶、外延、车规可靠性，而非 EUV 光刻；需求由新能源（电动车 / 光伏 / 储能 / 快充）与 5G / 数据中心供电驱动，是与 AI 主线并行、且国产替代推进相对顺利的一块。研究重点：衬底自给率、车规导入（上车）进度、产能扩张与价格走势。

用探针扎每一颗芯片测好坏。产出：标好良品的晶圆(良率)。由代工厂 / 测试厂做。
⑥封装
减薄 → 切割成裸晶 → 贴装键合 → 塑封做出引脚/焊球。产出：封装好的芯片。由封测厂(OSAT)做。 ↪ 流程详见 后道·封装测试 与 先进封装。
⑦成测 & 出货
成品测试、打标、出货分销。产出：可用的成品芯片。由封测厂、分销商做。

关于"硅料"的一个常见混淆
多晶硅(硅料)分两档：① 太阳能级(纯度约 6~9 个 9)给光伏发电用；② 电子级(纯度要到约 11 个 9，即 99.999999999%)才是造芯片用的，纯度高一个量级、是另一个小得多的赛道。平时新闻里说的"硅料涨价"多半指太阳能级，和芯片用的电子级不是同一种货。
